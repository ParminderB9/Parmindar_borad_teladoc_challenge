Prerequisite:
1> Install Python
2> Install Selenium
3> Install Behave


To run Add user test:
1. Open terminal window
2. Go to project path "Parmindar_borad_teladoc_challenge/features/"
3. Type "behave add_user.feature"

To run Delete user test:
1. Open terminal window
2. Go to project path "Parmindar_borad_teladoc_challenge/features/"
3. Type "behave delete_user.feature"
